# htmlcss-responsive-layout

## Description
This project is a responsive website designed using HTML and CSS, following a mobile-first approach.

## Sections

### Header
The header includes a navigation bar with a logo on the left and a hamburger icon on the right for mobile and tablet views. On desktop, the hamburger icon is replaced with a menu.

### Main Sections

#### Section 1
Demonstrates key features with cards.
Responsive layout for both small and large screens.

#### Section 2
Highlights strategic planning and app development.

#### Section 3 (Testimonials)
Displays customer testimonials in cards.
Employs a violet color scheme for a cohesive design.

#### Section 4 (Benefits)
Lists the benefits of working with the company.
Each benefit presented in a card format with additional details.

#### Section 5 (About Us)
Provides information about the company.
Integrates a quote and a call-to-action button.

#### Section 6
Showcases the "GET STARTED" section with pricing details.
Encourages user interaction with a "Get Started" button.

### Footer
The footer contains information about the company, including a privacy policy link.

## Technologies Used
HTML
CSS
Font Awesome (for icons)
Google Fonts (PT Sans)
